# Metales Del Sureste Andaluz

Expertos en carpintería metálica, estructuras de aluminio, cerrajería y forja artística. Con más de 20 años de experiencia transformando el metal en piezas únicas de alta calidad.

## Tecnologías Utilizadas

Este proyecto está desarrollado con:

- **Vite**: Herramienta de construcción ultra-rápida.
- **TypeScript**: Programación robusta con tipos.
- **React**: Biblioteca líder para interfaces de usuario dinámicas.
- **Framer Motion**: Animaciones fluidas y premium.
- **Lucide React**: Iconografía moderna y minimalista.
- **shadcn/ui**: Componentes de alta calidad y diseño limpio.
- **Tailwind CSS**: Estilizado moderno y eficiente.

## Desarrollo Local

Para ejecutar el proyecto localmente, asegúrate de tener Node.js instalado.

```sh
# 1. Instalar dependencias
npm install

# 2. Iniciar servidor de desarrollo
npm run dev
```
